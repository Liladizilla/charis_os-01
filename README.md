## 🚀 What is Adizilla_01?

**Adizilla_01** is a custom operating system built entirely from scratch by a self-taught developer from Kenya. It features a real-time terminal interface, system integrity monitoring, neural event pipeline management, and an anomaly detection engine — all running from a TypeScript + C core.

This is not a tutorial project. This is a fully functional OS shell with real architecture behind it.

---

## ✨ Features

- **Integrity Matrix** — Real-time system integrity monitoring at the kernel level
- **Neural Event Pipeline** — Live event syncing and pipeline management
- **Persistent Anomaly Detector** — Monitors and flags system faults automatically
- **Node Management** — Supports multi-node architecture (currently 12 active nodes)
- **Shadow Mode** — Stealth operation mode for low-profile execution
- **Compat Engine** — Compatibility layer optimized for cross-environment support
- **OS Shell v4.5.2** — Full terminal interface with kernel uplink
- **Time Drift Monitoring** — Precision timing and drift detection (0.00ms accuracy)

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Liladizilla/charis_os-01.git

# Navigate into the project
cd charis_os-01

# Install dependencies
npm install

# Copy environment config
cp .env.example .env

# Start the OS
npm run dev
```
